# WeMD - 使用指南

## 🚀 快速开始

### 启动应用
```bash
cd apps/web
pnpm run dev
```

访问: **http://localhost:5173/**

## 📝 使用方法

### 1. 编辑 Markdown
在左侧编辑器输入 Markdown 文本,支持:
- 标题 (`#`, `##`, `###`)
- 加粗 (`**文字**`)
- 斜体 (`*文字*`)
- 代码块 (` ```语言 `)
- 列表 (`-`, `1.`)
- 引用 (`>`)
- 链接 (`[文字](URL)`)
- 图片 (`![描述](URL)`)
- 数学公式 (`$公式$`, `$$公式$$`)

### 2. 实时预览
右侧会自动显示微信排版效果,所见即所得。

### 3. 复制到微信
1. 点击右上角 **"复制到微信"** 按钮
2. 打开微信公众号编辑器
3. 直接粘贴 (Ctrl/Cmd + V)
4. 完成!样式完美保留

## 🎨 默认样式

当前使用的是**优雅简洁**的默认主题:
- 标题: 加粗,渐进式字号
- 段落: 行高 1.8,舒适阅读
- 代码: 浅灰背景,粉红色行内代码
- 引用: 左侧紫色边框,渐变背景
- 链接: 紫色下划线
- 表格: 斑马纹,圆角边框

## 🔧 快捷键 (计划中)

- `Ctrl/Cmd + B`: 加粗
- `Ctrl/Cmd + I`: 斜体
- `Ctrl/Cmd + K`: 插入链接
- `Ctrl/Cmd + Shift + C`: 复制到微信

## 📱 响应式支持

- **桌面端**: 左右分栏,最佳体验
- **平板端**: 上下分栏
- **手机端**: 编辑器固定高度,预览滚动

## ⚠️ 注意事项

1. **浏览器兼容性**: 建议使用 Chrome/Edge/Safari 最新版本
2. **剪贴板权限**: 首次复制时浏览器会请求权限,请允许
3. **数学公式**: 需要网络加载 MathJax,首次渲染可能较慢
4. **图片**: 目前需要使用外链图片,暂不支持本地上传

## 🎯 最佳实践

1. **先写内容,后调样式**: 专注于内容创作,不要过早纠结样式
2. **使用标题层级**: 合理使用 H1-H3,便于读者理解结构
3. **代码块指定语言**: 如 `javascript`, `python`,获得语法高亮
4. **图片使用 CDN**: 使用稳定的图床服务,确保图片长期可访问
5. **预览后再复制**: 确认排版效果满意后再复制到微信

## 🐛 常见问题

**Q: 复制后粘贴到微信,样式丢失?**
A: 确保使用 Ctrl/Cmd + V 粘贴,不要使用"粘贴为纯文本"。

**Q: 数学公式不显示?**
A: 检查网络连接,MathJax 需要从 CDN 加载。

**Q: 代码块没有高亮?**
A: 确保在代码块开头指定了语言,如 ` ```javascript `。

**Q: 移动端体验不好?**
A: 本工具主要为桌面端设计,建议使用电脑操作。
